--- layout: default title: Alika POS --- Alika Analytics Point-Of-Sales

This product implements scanning of machine readable identity cards and the forwarding of that data to a cloud based service of your choosing. This software is made available as open source software.

Product Functionality

The software installs as a Windows service and interfaces with a 3M CR100 Swipe Reader. Whenever a machine readable document (such as a: passport, identity card, or driver license) is scanned the serive detects the scan, forwards the identity data to a cloud service of your choosing, and notifies the operator via the accompanying Windows tray application with feedback about the scan results and cloud delivery results.

The product has the following components:

  • 3M CR100 Drivers
  • Windows Service
  • Windows Tray Application
  • Windows Console Application for troubleshooting
  • An installer which installs and activates the above components
